# Restaurant-Copilot


## 🎯 Sprint 1 Goal (thin vertical slice)

> **Deliver a working “table → waiter → kitchen” flow:**  
> Customers can open the menu, build and place an order on the surface computer; the assigned waiter accepts or rejects it on a tablet; the chef accepts/rejects items and marks them ready on a kitchen display; the system logs states; waiter login is enforced.
>


## How to run

1. **Docker installed** (Docker Desktop or Docker Engine) and working:

   * Run: `docker --version`
   * Run: `docker compose version` (or `docker-compose --version`)

2. Build and start everything:

   ```bash
   docker compose up --build
   ```

   (or `docker-compose up --build`)

3. Open the prototype UI:

   * Frontend: `http://localhost`
